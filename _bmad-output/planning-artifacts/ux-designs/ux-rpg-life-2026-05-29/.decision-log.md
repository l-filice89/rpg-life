## Visual weight & tone (2026-05-29 — Luca)

| Dimension | Choice | Notes |
|-----------|--------|-------|
| **Shell** | **B** | Clean but game-leaning — Hero XP bar, Focus pill, skill chips on quest rows |
| **Reward moment** | **B** (+ **C on Hero level-up**) | Full modal, moderate celebration normally; full-screen takeover only when Hero levels up |
| **Overall tone** | **C** | RPG first; lightweight through speed and flow, not visual austerity |
| **Dark mode** | **Follow system** | Respect `prefers-color-scheme`; no forced default |
| **Working mode** | **Coaching path** | Walk decisions; creative tools woven in |

**Design intent distilled:** Day-to-day surfaces feel like a quest log with readable game chrome. The dopamine beat is the Reward modal; Hero level-up is the rare epic moment worth full-screen treatment.

### Star Path layout traits (behavioral + visual)

- Generous vertical rhythm; quest rows breathe
- Hero zone: level label + **glowing gradient XP bar** (violet/teal)
- Focus pill: compact, secondary to Hero bar
- Quest rows: soft elevated surfaces, minimal border noise
- Overdue: neutral muted styling — no red alarm
- FAB: floating, accent-adjacent glow hint

### Motion (Luca: **B**)

- Reward modal appears promptly (≤1s NFR); no long intro animation
- Per-Skill XP bars **fill ~400ms** on Reward modal open
- **Subtle confetti** only on Hero level-up (not every complete)
- Hero level-up: escalates to **full-screen C treatment** + extended celebration
- Respect `prefers-reduced-motion`: skip confetti/fill; show final values instantly

### Skill visual identity (Luca: **B** — unified palette, confirmed)

- All 7 Skills share Crystal Path neutral chip styling (muted surface + foreground)
- Differentiation by **icon only** (Lucide):
  - Concentration → `Target`
  - Vitality → `HeartPulse`
  - Lore → `BookOpen`
  - Presence → `Users`
  - Order → `LayoutList`
  - Resolve → `Shield`
  - Craft → `Hammer`

### Auth UX (Luca: **B** — both screens)

- **Sign in:** RPG-framed — "Enter the realm", Crystal Path styling, one-line quest flavor
- **Post-send (same page, not a route):** "Check your stars" register, masked email, star motif, resend option — user then checks **email app** for magic link
- → `mockups/auth-sign-in.html`

### Quest board clear (Luca — updated 2026-05-29, revised)

- Shows **immediately after last open Quest is completed** — when Reward modal dismisses and open count = 0 (not deferred to refresh)
- Copy dual intent: **celebrate** ("Quest board clear") + **invite** next Quest ("Start another when you're ready." + primary **Add a quest** + FAB)
- Optional secondary: "See your growth" → My Profile
- Distinct from first-time zero-Quest empty ("No quests yet")
- Mock: `mockups/board-clear-empty.html`
